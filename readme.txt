== Description ==

Plugin Memorandum makes you can write memo or notes for each plugin in the Plugins page.

Custom Header designed by [Design Oil](http://www.design-oil.com/). Thanks a lot.

== Installation ==
1. Upload the Plugin Memorandum folder to the plugins directory in your WordPress installation
2. Go to plugins list and activate "Plugin Memorandum". 
3. You can write and save memo for each plugin in the Plugins page.
